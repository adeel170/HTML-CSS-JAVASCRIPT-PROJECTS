# 💳 SwiftPay

SwiftPay is a full-stack digital payment application that allows users to manage wallets, perform transactions, and access various financial services securely.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 👤 User Profile Management
* 💰 Wallet System
* 🔄 Transactions (Send / Receive Money)
* 🧾 Services Integration
* 🔒 Secure API with Middleware Authentication

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB (Database)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/adeel170/swiftpay-v2.git
cd swiftpay-v2
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
```

Create `.env` file based on `.env.example`

Run server:

```
npm start
```

---

### 3️⃣ Run Frontend

Open:

```
frontend/index.html
```

in your browser
